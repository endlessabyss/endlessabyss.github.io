在ES6中针对含税， =>代表函数

箭头函数使用规则

  1. 若函数没有参数，则不能省略（）

    ​ ①若函数体只有一句代码，则可以省略{}，

    ​ 若不是return语句，但是需要注意在语句前添加void；

    ​ 若是return语句，则直接省略return即可

    ②若函数体有多条语句组成，咋不能省略{ }

  2. 有参数函数

    ①有一个参数，可以省略（） 其余和以上类似

    ②多个参数，不能省略( )

    ③若函数体只有一句return语句，且返回值为对象，则使用（）包裹，若返回值为数组则不需要处理
